# 都结束了？Gemini 2.5 Pro免费API套餐暂停、Grok3 API 免费白嫖 150 美金活动即将结束

在过去的几个月里，Google 和 xAI 分别推出了 Gemini 2.5 Pro 和 Grok API 的免费套餐和优惠活动，也是吸引了大量开发者和 AI 爱好者的关注。

不过最近他们竟然不约而同地宣布取消这些优惠活动了...

# Gemini 2.5 Pro 免费套餐暂停

Gemini 2.5 Pro 是 Google 推出的最先进的 AI 模型之一，具备强大的推理和多模态处理能力。 最初，Google 在 Google AI Studio 提供了 Gemini 2.5 Pro 的免费试用版本，供用户体验其强大的功能。 但近期Google 的首席产品官发布，API中Gemini 2.5 Pro 的免费套餐已被暂停，用户将无法再通过免费渠道访问该模型。 虽然 Google 尚未发布正式公告，但这一变动引发了开发者社区的广泛讨论。

有些用户认为，这一举措可能是 Google 为了限制免费用户的使用量，确保付费用户的体验质量。 此外，部分用户指出，Google 可能希望通过限制免费套餐，迫使用户转向付费版本，另外官方也声称新客户可以在注册付费帐户后还可以获得 300 美元的免费 API 积分。

也有人猜想可能3.5即将发布，资源正慢慢倾斜。

对于开发这来说，Gemini 2.5 Pro 的免费API套餐结束了，但 Gemini Advanced 的教育邮箱薅羊毛还没有结束，晚了可能这个也要结束啦~
