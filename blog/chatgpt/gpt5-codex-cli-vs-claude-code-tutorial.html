Claude Code降智？来试试GPT-5-Codex！国内如何使用到 Codex CLI 教程

最近我做了个决定——暂别 Claude Code，全面转向 OpenAI 的 Codex。用下来就两个字：真香。

为什么？很简单，三个理由说清楚：

  • 理由一：政策与体验都不爽。 Claude 最近对“中国控股公司”的封禁政策让我很难受，使用层层受限，体验打折。
  • 理由二：账户割裂，成本浪费。 我长期用 ChatGPT。为了用 Claude Code，我得同时开两边会员，既贵又乱，这笔钱花得不值。
  • 理由三：高峰“降智”，口碑穿帮。 社区这段时间频繁反馈 Claude Code 高峰期切换“低成本模式”，体验忽上忽下——这我有点受不了。

换到 Codex 后怎样？一句话：不用看人脸色了。ChatGPT 会员里直接能用 Codex，CLI 终端、VS Code 插件、云端任务全覆盖，我的工作流顺到飞起。

可能有小白还清楚Codex是什么，有哪些优势，如何快速订阅以及相关额度是怎么样的，本次将一一讲解

Codex 到底是什么？

Codex 是 OpenAI 出品的“代理式编码（Agentic Coding）”工具

它不仅能读懂你的项目，还能修改/运行/测试，必要时把任务丢到云端自己拉依赖、配环境、跑结果，甚至和 GitHub 打通做审查与提 PR。

GPT-5 和 GPT-5-Codex 的区别

而在前不久，GPT-5-Codex 正式上线，在 GPT-5 基础上专门为“代理式编码”做了强化，既能秒回小活儿，也能长时间独立推进复杂任务。官方实测最长可连续工作 7+ 小时，小任务时还能明显减少消耗。

根据 OpenAI 的员工流量统计，在模型生成的令牌数量（涵盖隐藏推理和最终输出）中，位列用户使用量后 10% 的轮次里，GPT-5-Codex 的令牌消耗比 GPT-5 少了 93.7%。

三种典型使用方式

  • Codex Cloud（网页版）：把重活儿交给云端，长时间任务/跨仓重构/复杂依赖都不再是负担

  • Codex IDE扩展：装上 VS Code / Cursor / Windsurf 插件，无缝预览 Diff、理解上下文、多文件联动，写/改/测一气呵成。

  • Codex CLI：开终端就能用，诊断日志、修脚本、跑单测、做小改动，快进快出超高效。

一句话总结：终端搞定小事、IDE 做重构、云端扛长活——任你切换。

Codex 三种典型使用方式

一、Codex 网页版的使用

如果你是订阅了ChatGPT的话，在官方聊天页面中的左上角，我们可以看到Codex的快捷入口，这对于一些小白来说是可以直接省去一些安装授权等问题的。

Codex Cloud 需要授权Github对你的代码仓库做读写，目的也是为了方便源码的管理迭代。

如果已经有Github账号的话，可以直接登录授权；没有的话也没关系，直接用你的google邮箱创建一个即可~

在开始使用 Codex 前，需要先准备好一个“环境”。这里的环境，其实就是你在 GitHub 上的一个仓库。

如果你已经有 GitHub 项目，那直接选中即可；如果还没有，就去 GitHub 新建一个空仓库，建好之后回到 Codex 点一下刷新，新的仓库就会出现在列表里了。

当你选定好对应的环境之后，就可以正式开始编程了。

这里需要特别注意的是：如果你的仓库是公开仓库，那么任何人都是可以访问和查看其中的内容。

这种方式虽然快速无需安装，很适合小白，但代码始终是在云端沙箱上的，对于代码隐私还是有待考察的。

二、在 VS Code 里无缝用 Codex

  1. 打开 VS Code（或 Cursor / Windsurf）。

  2. 插件市场搜索 “Codex”，认准官方的，一键安装即可。

  1. 安装之后，你就可以在左上角点击进入 chat

个人习惯可以把聊天页面放置在右边会比较顺手一些

  1. 首次使用会提示登录，强烈建议用 ChatGPT 账号。

  1. 之后你就可以在聊天框中选择 GPT-5-Codex 模型进行对话、选择上下文、查看 Diff，边聊边改边预览。

目前 macOS、Linux 体验成熟；Windows 支持仍在打磨中（但我日常也能用）。

三、安装Codex CLI

Node/npm 安装：

npm install -g @openai/codex

macOS（Homebrew）：

brew install codex

验证版本：

codex --version

第一次运行会做登录授权：

codex

首次运行 Codex CLI 时，系统会自动弹出授权界面提示你进行身份验证。

授权之后即可进行使用

在当前你要进行 “AI协助” 的路径下输入Codex即可

建议直接用你的 ChatGPT 账号登录（Plus/Pro/Team/Edu/企业均可），目前免费用户无法使用到Codex。授权后就能用订阅额度，无需额外 API 费用。
