# OpenAI 正式全面开放GPT-4.1，代码能力提升，如何使用到GPT4.1呢？

OpenAI 近期的一系列发布着实令人瞩目，甚至可以说是惊喜连连。

先是传闻中性能强劲的 GPT-4.5 吊足了大家的胃口，紧接着，GPT-4.1 就以 API 的形式正式登场。就在今天凌晨的直播中，OpenAI 推出了全新的 4.1 系列，包含了三款模型：

  • GPT-4.1 (旗舰版)
  • GPT-4.1-mini (轻量版，主打高性价比)
  • GPT-4.1-nano (微型版，号称 OpenAI 史上最经济)

还记得之前那个引领AI热潮的GPT-4吗？GPT-4 自 2023 年 3 月发布以来，在长达一年多的时间里都是 AI 大模型领域的标杆，而前不久，OpenAI宣告它将逐步下线，而现在GPT-4.1的出现无疑是GPT-4的延续...

作为 GPT-4 的接棒者，GPT-4.1 被 OpenAI 寄予厚望。官方宣称，它在编码、长上下文处理、指令遵循以及视觉等多项任务上，均显著超越了此前的明星模型 GPT-4o。更令人惊讶的是，在某些关键指标上，GPT-4.1 甚至能够比肩乃至超越价格更高的 GPT-4.5！

OpenAI 自信地表示其“每个维度都更强”，底气十足。

我们来快速梳理一下 本次 GPT-4.1 的主要亮点：

  • 100 万 Token 上下文窗口：相当于约 75 万字的文本处理能力。

  • 卓越的编码能力：在 SWE-bench Verified 测试中得分高达 54.6%，比 GPT-4o 高出 21.4%，甚至比 GPT-4.5 强 26.6%。

  • 强大的多模态功能：支持文本、图像、视频输入，在 Video-MME“长视频无字幕”测试中取得了 72% 的准确率，远超 GPT-4o 的 65.3%。

  • 更精准的指令遵循：能够更准确地理解复杂指令，非常适合驱动 AI 代理执行任务。

  • 知识库已更新至 2024 年 6 月。

此次 GPT-4.1 特别突出了其在编码和长上下文方面的能力。从官方提供的两大编程基准测试结果来看：

在由 OpenAI 与原始作者合作开发、用于评估 AI 模型软件工程能力的 SWE-bench Verified (SWE-bench 数据集的精炼子集) 上，GPT-4.1 的得分超越了 o1-high、o3-mini 和 GPT-4.5。

相比SWE-benchVerified，只有Python，而且偏工程化场景，而 Aider 是一个覆盖 C++、Go、Java、JavaScript、Python、Rust 等 6 种编程语言的多语言编程测试集，更侧重于编程语言特性和复杂逻辑。

在此测试中，GPT-4.1 的表现也超越了 GPT-4.5，但略逊于 o1-high 和 o3-mini-high 这两款推理模型。

除了性能上的飞跃，GPT-4.1 另一大亮点便是其极具竞争力的价格（单位：美元）。

  • 比 GPT-4o 更经济：价格相较 GPT-4o 降低了 26% 至 83%。

  • 对 GPT-4.5 形成“价格优势”：性能上能与 GPT-4.5 抗衡甚至部分超越的 GPT-4.1，其价格竟然仅为 GPT-4.5 的约三十七分之一。

因为GPT‑4.1 在许多关键能力上提供了更好的或相当的性能，同时成本和延迟更低，GPT‑4.5 Preview API将于2025 年7 月14 日正式下线，这也表明，GPT-4.1 不仅是 GPT-4 的替代者，更是 OpenAI 未来一年主推的旗舰模型。

理论上的优势已经相当明确：性能大幅提升，价格大幅降低。

# 如何使用到GPT-4.1

目前GPT4.1已全面开放，GPT-4.1 的部分功能现在对免费用户开放。

OpenAI 已将 GPT-4.1 mini 模型引入 ChatGPT 的免费版本，它取代了之前的 GPT-4o mini，成为免费用户在达到 GPT-4o 使用限制后自动切换的备用模型。这意味着，即使不订阅 ChatGPT Plus，您也可以在一定程度上体验 GPT-4.1 的能力。

免费用户在 ChatGPT 中默认使用的是 GPT-4o，当达到 GPT-4o 的使用限制时，系统会自动切换到 GPT-4.1 mini，确保您可以继续使用聊天功能。

Plus、Pro、团队用户可以在模型选择处下拉，选择使用GPT-4.1；企业和教育用户将在接下来的几周内获得访问权限。

如果你希望获得更高的使用配额或访问完整版的 GPT-4.1，可以考虑订阅 ChatGPT Plus、Pro 或 Team 计划。
