# 【保姆级教程】国内如何注册和绑卡美区Apple ID，以及安装订阅 ios app 的 ChatGPT Plus，成功率100%

最近很多人见识过Grok3的能力之后，觉得还是ChatGPT比较好用，很多人下载安装了ChatGPT IOS版本，但苦于没有美区的Apple ID。

那美区Apple ID应该如何注册，如何绑卡，以及如何订阅IOS ChatGPT Plus等的呢？

求人不如求己，下面将教大家如何自己申请注册一个美区的Apple ID，成功率100%，且不需要任何代理工具。

# 1. 准备工作

  • 一个能接收短信的国内手机号

  • 一个全新邮箱（指从没注册过 Apple ID 的邮箱），目前 QQ 不行，但163 和 gmail 可行。

  • 不需要任何代理工具。

注意以下三点可以避免很多问题：

  • 出生日期：一定要设置成大于 18 周岁的日期，否则会导致部分应用由于年龄限制无法使用。

  • 电子邮件：建议新注册一个全新的从未注册过 Apple ID 的邮箱。目前QQ 不行，163 和 gmail 可行。

  • 密码：设置密码时，密码中不要包含有名字、生日、邮箱中的信息，否则会卡在验证码那一步。

# 2. 注册详细步骤

整个注册流程在手机上能操作，电脑上同样可行，不过从方便程度来讲，还是建议用电脑来弄。

# 2.1 创建 Apple ID

首先进入美国 Apple ID 注册页面，复制 https://appleid.apple.com/account 至 Safari浏览器 中打开即可进入。

有可能你会遇到如：目前无法完成你的请求-此时无法创建你的账户。

这个是官网检测你的网络状态不正常。

可以多试几次或换个网，或者修改你的设备的默认语言和地区设置，即可通过。

手机上是注册显示 目前无法完成你的请求-此时无法创建你的账户, 可以修改语言和地区。

# 2.2 电子邮件+手机号码 进行验证

接下来需要对 电子邮件和手机号码 进行双重验证。

把收到的验证码输入进去，依次点击 下一步 就行了。

此时美区apple id已经注册成功，只是还没有填写支付方式， 没有激活，不要慌，我们切换到手机上去激活刚注册的美区Apple ID 。

# 2.3 修改个人信息

注册成功后，账号会自动登录，如果没登录的话重新登录一遍刚注册的账号即可。

然后点击 个人信息，接着再点击 国家或地区。

# 2.4 更改国家并填写 WildCard 卡片信息

修改国家之前，我们需要绑定付款方式。这里如果没有国际visa、master卡的话，可以直接到wildcard注册一张虚拟master卡。

wildcard官网：https://yeka.ai/i/GPTSHOW

注册申请之后就可以获得一张类似master的虚拟卡，卡费2年10.99刀。（现在卡费降低，还记得我当时1年是9.99刀）

在添加付款方式的页面里，把国家或地区更改成 "漂亮国"，付款方式按标注的提示。填写在 WildCard官网中申请的消费卡的具体卡号，有效期，CVV和账单地址即可。

Expiration Date （有效期） 和 CVV 在卡上的对应位置在这里：

**对于电话号码，一定要填，但不用接收验证码，所以你可以随便填。 **

建议你确保前3位数字与你帐单地址的区号匹配，后7个数字随意。

  • Delaware—-302
  • Montana—-406
  • Oregon—-541
  • New Hampshire—-603

如果不知道怎么填，也可以访问这个 生成随机M国手机号的网站，复制一个随机生成一个手机号，即可填写。

https://codebeautify.org/random-phone-number

最后就可以点击【更新】, 点击更新 Save Changes，这个时候 你的 Apple ID 已全部设置为美区。

# 2.5 再次检查卡信息

转到“付款和配送”，卡片已添加成功，不过还需要再次验证下卡片信息，点击卡片输入 CVV ，然后保存修改即可。

到这里，你的美国 Apple ID 就可以使用了。

下面教大家如何登录美区 ID：

# 3. 登录你的美区ID

打开 App Store ，首先退出当前账号。点击右上角的头像，然后拉到末尾，点击 退出登录 即可。

再次点击 【App Store】 中的右上角的头像，输入前面注册的美区账户和密码，然后登录。

此时，你应该能在付款方式下看到你的绑定成功的卡。

# 4. 下载 ChatGPT 应用程序并订阅 ChatGPT Plus

在 App Store 中搜索 ChatGPT 进行下载:

或者访问下面网址进行下载

https://apps.apple.com/us/app/chatgpt/id6448311069

打开 ChatGPT 应用程序，点击左上角的菜单，然后选择“设置”。

点击“升级到 ChatGPT Plus”，然后按弹出窗口底部的“订阅”按钮进行订阅。

注意：升级前，记得给自己的虚拟master卡进行充值余额不低于20刀。 可以使用支付宝的方式进行充值。

如果你还有什么不懂的话，可以前往官网查看视频

官网：https://yeka.ai/i/GPTSHOW
