# WildCard虚拟卡的常见问题解答

#

Q1：WildCard 能做什么？有哪些工具和功能？

A1：WildCard，就是为了解决大家使用海外 AI 产品而生的，平台针对国内用户支付海外服务常见问题也给出了很不错的解决方案

  • 痛点一：国内信用卡无法跨境支付 平台针对性提供美国虚拟借记卡，支持支付多种海外服务。

  • 痛点二：多人混用同一网络出口，被判别为盗刷，无法成功支付 平台提供专用的美国家庭网络出口，确保支付成功，支付不成功全额退款

  • 痛点三：注册海外服务门槛高，需要境外手机号和邮箱 平台订阅海外服务操作简单，同时还提供海外手机号和海外邮箱，方便注册海外服务

#

Q2：Wildcard开卡如何收费？

A2：WildCard 开卡可以选择 2 年或 3 年服务时间。
2 年的年费是 11.99 美元。
3 年的年费是 16.99 美元。

如果填入平台专属邀请码"GPTSHOW"

或者直接点击链接👇 https://chatgptshow.com/bewildcard.html

即可享受官方开卡减免优惠！！！

#

Q3：充值消费有手续费吗？

A3：消费手续费 0%，但非美国商家/非美元的消费手续费 1%+$0.5，比如您消费英镑等其他货币。

充值手续费是 3.5%，充值的余额是可以实时提现的，提现手续费 0%。

WildCard 支持退款到卡上，如果您跟商家协商退款，一般 14-21 个工作日退款到卡上。

退款会收取2%手续费。

绑卡的时候，一般绑卡平台会进行小额扣款验证，然后撤销这笔扣款，撤销这笔扣款时，会产生 1 美元的撤销费用，如果扣款不足 1 美元则不会收取。

如果商家撤销这笔交易，进行退款的时候，同样也会收取这笔1 美元的交易撤销费。

#

Q4：海外服务有哪些是支持的，哪些是不支持？

A4：支持的服务有很多，列出最常问的：
ChatGPT、OpenAI API、Apple Store、Claude、MidJourney、Google Play、Google Gemini、Github、Steam美区、Giffgaff(看网络质量)、AWS(需自己解决地址证明)、Telegram、Cursor、Amazon、Poe、Twitter、Facebook...

不支持的服务:
Google Cloud Platform、Oralce Cloud、Netflix/Hulu/Spotify(媒体类)、Paypal/Wise (银行类)、Blizzard/EA 游戏类、各类土耳其/阿根廷区支付

#

Q5：WildCard 能收款/转账吗?

A5：只能消费，不能收款和转账。

#

Q6：WildCard 不充值就无法使用吗？

A6：是的，因为 WildCard 本质上是一种虚拟借记卡，余额为 0，不充值是无法扣费的，请务必在充值后再进行绑定/支付。

所有的海外服务绑卡都有小额扣款验证，如果卡内余额为0，是无法通过的，还会因为余额不足消费失败。累计十次会销卡，为了保护您卡片不被注销，我们做了这个限制。

#

Q7：WildCard 2年有效期之后会失效吗？

A7：有效期（2 年）内均不会过期。

2年有效期之后，可以续费 $9.9 美元继续开一张 2 年有效期的新卡。

如果不想续费，余额会被自动退回到充值账户。

#

Q8：填写卡号时出现的 ZIP 该填什么？

A8：大家绑定支付方式时，有时候会碰到填写卡号时，也需要填写 ZIP。

ZIP 指的就是您账单地址所在的邮编，也就是 Postal Code，直接填写账单地址的 Postal Code 那 5 位数字就可以了。

#

Q9：WildCard 能开发票吗？

A9：平台是境外实体，可以给您开具能在国内报销的开卡收据和月结单。开具入口在登录之后的右上角「设置」中。

生成的收据包含当月所开的开卡费以及充值产生的手续费。OpenAI API 的收据在 Settings-Billings-Billing history

#

Q10：虚拟卡的开卡费能退吗？

A10：退不了，因为 WildCard 的开卡好似你在银行开的卡一样，开卡费不能退，但是冲的余额可以退到支付宝的。
