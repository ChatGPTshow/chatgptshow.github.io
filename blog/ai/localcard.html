# Localcard是什么？

官网介绍：https://web.localcard.hk/

秉承着合规、精品、风控、开放、有所为有所不为的发展理念，Localcard致力于为全球企业提供便捷、快速、稳定的灵活用工支付解决方案。

采用领先的多重签名机制（冷/热钱包、智能合约），确保用户资金的安全性，所有交易数据均采用先进的加密技术处理，以防止外部攻击与数据泄露风险。

携手发卡机构推出的虚拟银行卡/实体信用卡，支持Visa/MasterCard线上直接付款。审核简单、申请快速、可解决跨境多场景支付难题。

客户范围覆盖游戏、短视频、社交平台等多领域

# Localcard业务场景

支持各类海淘消费，游戏，软件，AI等海外业务消费

拥有全球代付能力，面向全球不同国家或地区，支持多币种支付

# Localcard平台优势

# 一、虚拟卡优势

# 支持多卡段，覆盖广泛场景

Localcard支持多个主流卡段（4411、4040、5533、5190、6587、5592、5257、5258），意味我们支持的场景多，覆盖市场上99%的应用场景，满足各类支付需求。

# 费用透明，性价比高

充值手续费低至1%，开卡费用低至8刀，并可使用优惠券享受更低成本（5美元优惠券代码可见文末获取）。

# 卡片级别丰富，适配多样需求

针对需要Visa或MasterCard的不同消费群体提供多种选择，用户可申请不同的虚拟卡，每个账号可拥有多张虚拟卡。

# 低费率优势

5561/6587 卡段免跨境手续费，免月费。

5533、5190、6587、5561、5257、5258卡段均免月费。

# 特殊功能支持

用户可将虚拟卡绑定至其他支付渠道（wechat pay， alipay等）或者消费场景（美团，淘宝等）。

5258卡段可绑定Apple Pay，目前市场比较紧张

5319卡段支持绑定Twitter（X）、OnlyFans及Telegram等

# 兼容多种支付平台

所有卡段支持绑定支付宝，部分卡段支持绑定微信。

全部卡段均可支持OpenAI相关服务。

# 二、实体卡服务

# 实体卡U卡专门为币圈服务

可绑定支付宝、微信，方便主流支付平台使用。 支持ATM取现功能，具备便捷性和高可用性。 支持NFC功能，且费率低，极具竞争力。

# 三、全球代付服务

# 多平台代付支持

提供支付宝（Alipay）、PayPal等电子钱包的代付服务。

面向全球用户，支持多种海外场景，多种支付渠道和需求。

# 开卡须知

  1. 注册后来到最右栏"我的"->"安全设置"，设置KYC认证和资金密码（KYC认证仅针对实体卡用户，虚拟卡用户不需要认证）

  1. 充值时会出现三个网络，这些网络的手续费各不相同，推荐使用 BEP20 中间的网络，手续费最低，大部分交易所或冷钱包是支持TRC20的，ERC20手续费较高，一般不推荐。

  1. 虚拟卡目前的级别及开卡费用
  • 白金卡(13 USD)
  • 钻石卡（30 USD）
  • 铂金卡(20 USB)
  • 黑金卡（50 USB）
  • 普通卡（5 USD）

每种卡都有自己不同的消费场景、用卡权益、卡段，卡段可在该级别卡下左右滑动切换

可根据卡片特色、支持消费场景自由选择卡段，用卡须知请自行查看

  1. 实体卡

  1. 代付
