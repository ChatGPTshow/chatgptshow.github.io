CursorWildcard

Cursor AI IDE 编辑器的应用正在迅速普及，不仅在程序员群体中备受欢迎，产品经理、设计师，甚至其他非技术领域的从业者也纷纷尝试使用。许多非技术背景的用户通过 Cursor 顺利完成了自己的第一个程序或网站，可见其门槛之低和工具之强大。

交流群中也有小伙伴们通过Cursor实现了自己应用程序的快速代码编写。

Cursor 的 @CodeBase、Composer 等功能十分强大，尤其是在配合配置好的 cursorrules 文件时，用户仅需通过自然语言对话即可完成约 90% 的工作任务。这种人机交互的工作方式，极大地提升了开发和创作效率，为不同领域的用户带来了更高的生产力。

下载安装 Cursor 非常便捷，用户可以直接访问其官网获取安装包。

值得注意的是，免费版用户可以试用两周的 Cursor Pro 功能，但如果需要继续使用高级功能，则需订阅 Cursor Pro 或企业版。这种模式让用户可以先充分了解其价值，再决定是否升级，非常人性化。

# 如何订阅升级 Cursor Pro

由于 Cursor 普通版有限制，不仅使用时间短，还限制了你的请求数量，所以需要订阅升级 Cursor Pro 才能享受更快且无限次的高级请求，此外它还支持每天 10 次的 Claude Opus 使用。

Cursor Pro的订阅主要还是解决国内无法进行支付的问题。 海外大部分软件服务，都是不支持国内的支付方式。除非你有国际的visa、master卡。

接下来，将一步步教你如何 5分钟 在国内仅通过支付宝获取一张可长期稳定使用的国外虚拟信用卡。

# 订阅步骤

（1）打开Cursor官网：https://www.cursor.com/

登录自己的账号之后，点击头像 -> Account Settings。

（2）根据需求选择升级为Cursor Pro 或企业版，点击之后，就可以跳转到Cursor Pro 的订阅页面。

（3）进入到订阅页面后，保留页面，不关闭。去申请一张虚拟卡进行绑定。

(4)访问WildCard国内虚拟卡官网，并点击右上角的“登录/注册”按钮。

WildCard官网：https://yeka.ai/i/GPTSHOW

(或者直接点击文末"阅读原文"直接跳转)

(5) 填写 手机号、验证码，然后「提交注册」👇：

（大陆地区 选择 + 86 开头的区号，香港地区 选择 + 852）

(6) 开卡可以选择 2 年 或 3 年 有效期，可按自身需求选择年限，使用 平台专属邀请码 可以最大减免开卡费用，很多人就是不知道这个，导致贵了很多。

（7）再次点击 "我的卡片"，即可查看虚拟卡的卡片信息

（8） 点击“充值”按钮，并按照提示使用支付宝进行充值支付。

（9）在【左侧边栏】，找到并选择“Cursor 一键升级”

（10）现在回到【Cursor的订阅页面】之后，然后全选（Ctrl+A）上面的链接，进行复制(Ctrl+C)。

(11)回到 wildcard 中把复制的订阅链接，直接放入到输入框内，点击一键升级即可。

上面是一键升级到方式，对小白来说比较友好。当然了，你可以直接拿卡号信息直接绑定，然后进行支付也是可以的。

整个流程大概10分钟左右就可以完成升级，后续需要用到的话可以用这种方式订阅海外服务。
