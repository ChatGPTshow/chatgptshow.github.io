# Grok 2 和 Grok 3 使用教程：教你如何获得Grok3的访问权限，内附Grok3订阅升级订阅高级会员教程

近期，Grok3已发布，很多人都有这样的疑问：

Grok3 网页版地址在哪里访问，怎么安装使用订阅Grok3等

本文将分为两部分：Grok 2 和 Grok 3

先介绍如何访问 Grok 2，再进行介绍如何获得 Grok 3 的访问权限。

如果你知道如何访问Grok的话，可以直接看 Grok 3 如何使用访问部分。

# 访问Grok 2 教程

目前，Grok 2 的网页版在英国和欧盟地区无法直接访问，但您可以通过 iOS 设备上的 App Store 进行安装。

在开始之前，让我们了解一下 Grok 的几个官方平台: 网页版、API平台 和 移动端。

# （1）Grok 网页版

Grok 的网页版目前仅在美国、澳大利亚和印度地区可用。因此，要访问 Grok，您需要使用一些网络工具。

web网页版：https://grok.com/

您还可以在 X（原推特）平台上使用 Grok。

X （原推特）上的网页版：https://x.com/i/grok

# （2）Grok API 平台

API平台：https://console.x.ai

# （3）Grok 移动端

目前，Grok 仅支持 iOS 系统，您可以在 iPad 或 iPhone 上的 App Store 搜索并安装。

目前还没有桌面端应用，所以你用Mac电脑是搜索不到的。

如果您只是想试用 Grok，建议直接使用网页版，方便快捷。如果觉得不错，再安装 APP 也不迟。

# Grok 3 访问教程

重要提示： Grok 3 仅对 X 平台上的 Premium+ 用户开放。要使用 Grok 3，您需要订阅 X 的 Premium+ 服务。

也就是说，你需要注册一个X账号（原推特）。并且升级到对应的版本才能有权限访问 Grok 3。

# （1）国内订阅X的Premium+教程

首先，您需要注册一个 X 账号，可以通过 Google 邮箱进行注册。

X（原推特）官网：https://x.com/

进入官网后，点击侧边栏的“Premium”进入订阅页面。
